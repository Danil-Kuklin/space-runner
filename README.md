## Геймплей

Космический корабль движется вперед. Сверху летят метеоры. Цель - выжить как можно дольше и набрать максимум очков.

### Управление
| Устройство | Действие |
|---|---|
| Телефон | Касание и движение пальцем по экрану |
| ПК | Движение мышью по области игры |

---

## Механики

### Метеоры
- Летят вертикально сверху вниз с нарастающей скоростью
- Каждый пролетевший мимо метеор даёт +1 очко
- Столкновение -> конец игры, если щита не было 

### Сложность
- Скорость и частота метеоров растут плавно по времени и счёту

### Щит
- При подборе активируется на 3 секунды, вокруг корабля появляется золотая аура с таймером
- Поглощает одно столкновение с метеором, если ударился, то щит рассыпается

### Бонус звезды
- Начинают появляться после достижения 150 очков
- При подборе добавляют +20 очков

### Испытание - начиная с 250 очков
- Запускается при достижении TRIAL_SCORE_TRIGGER = 250
- 7 последовательных волн метеоров в случайном порядке (паттерны 0-6)
- Каждая волна - заранее рассчитанный ряд метеоров с безопасным проходом для корабля
- Обычный спавн метеоров на время испытания приостанавливается

## настройка констант

| Константа| Значение по умолчанию | Описание |
|---|---|---|
| SHIELD_DURATION | 3.0 | Время действия щита (сек) |
| SHIELD_SPAWN_INTERVAL | 12 | Интервал спавна щита (сек) |
| SCORE_STAR_VALUE | 20 | Очков за звезду |
| SCORE_STAR_SPAWN.minScore | 150 | Порог появления звёзд |
| TRIAL_SCORE_TRIGGER | 250 | Порог запуска испытания |
